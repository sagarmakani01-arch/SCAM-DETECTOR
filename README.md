## Overview

Nexar Verify is an AI-powered procurement verification platform designed to help organizations automate one of the most time-consuming business processes—verifying invoices, purchase orders, vendor information, and procurement documents.

Traditional verification workflows rely heavily on manual review, making them slow, expensive, and prone to human error. Nexar Verify streamlines this process by intelligently analyzing uploaded documents, extracting key information, detecting inconsistencies, and generating verification insights within seconds.

Whether you're a startup, SME, or enterprise, Nexar Verify helps improve operational efficiency, reduce financial risk, and build trust in procurement workflows.

---

# Why Nexar Verify?

Organizations process hundreds or even thousands of procurement documents every month. Manual verification introduces several challenges:

- Time-consuming document reviews
- Human errors during validation
- Duplicate invoice payments
- Fraudulent or manipulated invoices
- Missing or inconsistent vendor information
- Delayed procurement approvals
- Increased operational costs

Nexar Verify addresses these challenges with an intelligent AI-driven verification pipeline.

# Installation

Clone the repository

```bash
git clone https://github.com/<your-username>/nexar-verify.git
```

Navigate into the project

```bash
cd nexar-verify
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
